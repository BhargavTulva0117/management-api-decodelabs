🐄 Aureus Nutricare Product Management API
📖 Description
Aureus Nutricare Product Management API is a RESTful backend application developed using Node.js and Express.js. The API is designed to manage veterinary and animal healthcare products by providing endpoints for product management, customer inquiries, and newsletter subscriptions.
This project was developed as part of a Full Stack Development Internship to understand backend development concepts, API design, request handling, data validation, and CRUD operations.
🚀 Features
- View all products
- View product details by ID
- Add new products
- Update existing products
- Delete products
- Submit customer inquiries
- Newsletter subscription system
- JSON request and response handling
- Input validation
- HTTP status code management
- RESTful API architecture
-  🛠️ Technologies Used
- JavaScript
- Node.js
- Express.js
- Thunder Client / Postman
📡 API Endpoints
Product Management
```http
GET /products
GET /products/:id
POST /products
PUT /products/:id
DELETE /products/:id
```
Customer Services
```http
POST /contact
POST /subscribe
```
